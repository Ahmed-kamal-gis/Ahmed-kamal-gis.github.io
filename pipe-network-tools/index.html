About the Tool

The Problem

Preparing GIS infrastructure data to meet NWC schema requirements was a slow, repetitive manual process. Each project required classifying and reformatting large datasets — a task prone to inconsistency and human error when done by hand.

The Solution

Built a custom Python tool using ArcPy that automates classification, field mapping, and data formatting to match NWC's required GIS schema. What previously took significant manual effort now runs automatically — consistently and correctly every time.

Technical Details

What the Tool Does

  • Automatically classifies GIS assets by type and attribute
  • Maps and formats field values to match NWC GIS schema
  • Validates data consistency across large infrastructure datasets
  • Outputs submission-ready datasets without manual intervention

Development Highlights

  • Built entirely in Python using the ArcPy library for ArcGIS Pro
  • Designed to handle diverse asset types across different project scales
  • Improved productivity and reduced delivery time on GIS projects
  • Applied across multiple real NWC infrastructure projects
